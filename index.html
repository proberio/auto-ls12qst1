Uma pseudo-classe, em CSS, é uma palavra-chave adicionada a um seletor que especifica um estado especial para seleção de um ou mais elementos. Um exemplo clássico é a pseudo-classe “:hover”, associada a um link ou botão e que ativaria as regras sobre esses elementos quando o ponteiro do mouse fosse colocado sobre um destes elementos. Quais das opções abaixo usam classes que permitem trabalhar selecionando elementos para aplicar regras baseados em Foco do Elemento, Ativação do Elemento pelo usuário ou seleção de Elemento Requerido? (1 pto)

A) input:required { border-color: #800000; border-width: 3px;}

B) .blue-input:focus {background: yellow; color: blue;}

C) a:active { color: red;}

D) input:checked + label { color: red;}


Respotas

B que usa a regras focus e C que usa a regra active
